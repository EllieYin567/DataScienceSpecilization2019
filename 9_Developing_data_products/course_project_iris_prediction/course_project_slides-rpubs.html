ML prediction of iris species

Ellie Yin
2020-03-20

Background

This Shiny app will help you predict iris species

  • Training data: iris data in R
  • User input: Sepal.Length, Sepal.Width, Petal.Length and Petal.Width
  • Output data: iris species based on the model users choose

ML models

Two machine learning methods can be applied:

  • Multinomial linear classification
  • Random forest

App Output

The app will tell you which iris species it is based on your input data
